# 5. 자료구조
## 5.1 리스트
### 5.1.1 리스트 생성하기
- 형식 : 리스트명 = [값1, 값2, ...]
    - 빈 리스트 : 리스트명 = []
- 조회 : 리스트명[인덱스]

### 5.1.2 값 추가/삽입/삭제하기
- 인덱스로 리스트에 저장한 데이터의 위치를 표시
- index("문자열") : 인덱스 위치를 표시 
- append(추가할 값) : 리스트 끝에 값을 추가
- insert(인덱스, 삽입할 값) : 인덱스 위치에 값 추가
- pop() : 리스트 끝에서부터 값을 하나씩 꺼내어 반환 후 삭제
- clear() : 모든 값을 삭제

### 5.1.3 중복 값 확인하기
- count("값") : 동일한 값이 몇개 있는지 확인

### 5.1.4 리스트 정렬하기
- sort() : 오름차순으로 정렬
    - sort(resverse=True) = 내림차순으로 정렬 
- resverse() : 숫서 뒤집기

### 5.1.5 리스트 확장하기
- 리스트.append(리스트) : 서로 다른 리스트를 합칩

## 5.2 딕셔너리
- 딕셔너리 : 값이 key와 value 한 쌍으로 이루어져 있음

### 5.2.1 딕셔너리 생성하기
- 형식 : 딕셔너리명 = {key1: value1, key2: value2, ...}
    - 빈 딕셔너리 : 딕셔너리명 = {}
- key는 중복을 허용하지 않음으로 유일한 값으로 설정하여야 하며 변하지 않는 값을 사용하여야 함.
- 딕셔너리명[key]형태로 value에 접근 가능
    - 값이 없는 경우 오류 발생
- get(key, default=None) : key에 해당 하는 value를 조회 
    - key에 해당되는 값이 없는 경우 기본값 None 리턴, default 값을 설정한 경우 default값 리턴
- in : 딕셔너리에 key 값이 있는지 확인 혹은 문자열에 해당 글자가 포함되었는지 확인
    - 값이 있는 경우 True, 없는 경우 False

### 5.2.2. 값 변경/추가/삭제하기
- 딕셔너리[key] = value : key가 있는 경우 key 해당 되는 value 수정, key가 없는 경우 key 및 value 추가
- del 딕셔너리[key] : key 및 key에 해당 되는 값 삭제  

### 5.2.3. 값 확인하기
- key() : 딕셔너리에 있는 key 확인
- value() : 딕셔너리에 있는 value 확인 
- item() : 딕셔너리에 있는 key, value 둘다 확인
- clear() : 값 전체 삭제

## 5.3 튜플
- 튜플 : 리스트와 비슷하나 값의 변경이나 추가, 삭제 등이 불가능
- 형식 : 튜플명 = (값1, 값2, ...)
- 조회 : 튜플명[인덱스]
- 변수를 여러 줄에 따로 정의하는 대신 튜플 형태로 한 줄에 여러 변수 정의 가능함을 변수의 순서를 손쉽게 바꿀 수 있음
    - 형식 : (변수명1, 변수명2, ...) = (값1, 값2, ...)

## 5.4 세트
- 세트 : 중복을 허용하지 않고 데이터 순서가 없음
- 형식 : 세트명 = {값1, 값2, ...} or 세트명 = set([값1, 깂2, ...])]
    - 빈 세트 생성 : 세트명 = set()
- & 기호 또는 intersection() : 교집합
- | 기호 또는 union() : 합집합
- - 기호 또는 diffrence() : 차집합
- add() : 데이터 추가
- remove() : 데이터 삭제

## 5.5 자료구조의 변경
- type() : 자료구료를 확인
- list() : 자료형를 리스트으로 변경
- tuple() : 자료형을 튜플로 변경
- set() : 자료형을 세트로 변경

## 5.6 실습 문제 : 당첨자 뽑기
- suffle() : 리스트의 데이터를 무작위로 섞어 줌.
- sample() : 리스트에서 원하는 갯수만큼 중복 없이 뽑는 동작을 수행
