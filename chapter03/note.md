# 3장 연산자

## 3.1 연산자의 종류
- 연산자 : 프로그래밍에서 연산할 때 사용하는 기호
### 3.1.1 산순 연산자
- 산술 연술자 : 수를 연산하는 데 사용하는 사칙연산 기호

    | 연산자 | 의 미 |
    |-------|--------|
    |+|연산자 왼쪽과 오른쪽 더하기|
    |-|연산자 왼쪽과 오른쪽 빼기|
    |*|연산자 왼쪽과 오른쪽 곱하기|
    |/|연산자 왼쪽과 오른쪽 나누기|
    |**|연산자 왼쪽과 오른쪽만큼 거듭제곱|
    |%|연산자 왼쪽과 오른쪽을 나눈 나머지|
    |//|연산자 왼쪽과 오른쪽을 나눈 몫|
- 나누기 연산을 하면 결과 같이 소수점을 포함한 실수로 나옴.

### 3.1.2 비교 연산자
- 비교연산자 : 값을 비교할 때 사용하는 부등호
    | 연산자 | 의 미 |
    |-------|--------|
    |>|연산자 왼쪽이 오른쪽보다 큼|
    |>=|연산자 왼쪽이 오른쪽보다 크거나 같음|
    |<|연산자 왼쪽이 오른쪽보다 작음|
    |<=|연산자 왼쪽이 오른쪽보다 작거나 같음|
    |==|연산자 왼쪽과 오른쪽이 같음|
    |!=|연산자 왼쪽과 오른쪽이 다름|

### 3.1.3 논리 연산자
- 논리연산자 : 여러 조건을 결합해 참인지 거짓인지 판단할 때 사용
    | 연산자 | 의 미 |
    |-------|-------|
    |and|연산자 왼쪽과 오른쪽이 모두 참이면 참|
    |or|연산자 왼쪽과 오른쪽 중 하나라도 참이면 참|
    |not|값이 참이면 거짓, 거짓이면 참| 
- 단축 평가
    - a>b>c 는 a> b and b>c와 같음
    - and 와 or는 앞의 연산 결과에 따라 뒤의 연산이 수행되지 않을 수 있음

## 3.3 변수로 연산하기
- 복합 대입 연산자 : 대입 연산자와 산술 연산자를 합친 것
    | 연산자 | 의미 |
    |--------|-----|
    | += | 연산자 왼쪽 값에 오른쪽 값을 더한 후 왼쪽에 값에 대입 |
    | -= | 연산자 왼쪽 값에 오른쪽 값을 뻰 후 왼쪽에 값에 대입 |
    | *= | 연산자 왼쪽 값에 오른쪽 값을 곱한 후 왼쪽에 값에 대입 |
    | /= | 연산자 왼쪽 값에 오른쪽 값을 나눈 후 왼쪽에 값에 대입 |
    | **= | 연산자 왼쪽 값에 오른쪽 값을 거듭제곱 후 왼쪽에 값에 대입 |
    | //= | 연산자 왼쪽 값에 오른쪽 값을 나눈 후 몫을 왼쪽에 값에 대입 |
    | %= | 연산자 왼쪽 값에 오른쪽 값을 나눈 후 나머지를 왼쪽에 값에 대입 |

## 3.4 함수로 연산하기
### 3.4.1 숫자처리 함수
| 함수 | 의미 |
|------|-----|
| abs(x) | x의 절대값 |
| pow(x,y) | x를 y만큼 거듭제곱한 값|
| max() | 가장 큰 값 |
| min() | 가장 작은 값 |
| round(x,d) | x를 반올림한 값, d는 표시할 소수점 이하 자릿 수. d가 없으면 소수점 이하 첫째자리에서 반올림한 정수 |

### 3.4.2 math 모듈
| 함수 | 의미 |
|------|------|
| floor() | 내림 |
| ceil() | 올림 |
| sqrt() | 제곱근 |

- math 모듈을 사용하려면 모듈을 가져와야함
    - from math import * or import math

### 3.4.3 random 모듈
- random() : 무작위로 숫자를 뽑아주는 함수
    - 0 이상 1 미만(1은 불포함)에서 난수를 뽑는 기능
    -   | 함수 | 의미 |
        |------|-----|
        | randrange(시작숫자, 끝숫자) | 주어진 범위 안에서 정수인 난수 생성(끝 숫자 미포함)
        | randint(시작숫자, 끝숫자) | 주어진 범위 안에서 정수인 난수 생성(끝 숫자 포함)
