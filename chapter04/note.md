# 4. 문자열 다루기
## 4.1 문자열이란
- 문자열 : 문자들의 집합
- 작은 따옴표 또는 큰 따옴표로 감쌈  
- 여러줄 작성시 여는 따옴표와 닫는 따옴표는 각각 3개씩(큰따옴표 또는 작은 따옴표) 넣어 앞뒤로 감싸면 됨

## 4.2 원하는 만큼 문자열 자르기 : 슬라이싱
- 슬라이싱 : 원하는 데이터를 자름
    - 형식 : 변수명[시작 인덱스 : 종료 인덱스]
    - | 형식 | 의미 |
        |------|-----|
        | 변수명[ :종료 인덱스 ] | 처음부터 종료 인덱스 직전까지 슬라이싱 |
        | 변수명 [시작 인덱스: ] | 시작 인덱스부터 끝까지 슬라이싱 |
        | 변수명[ : ] | 처음부터 끝까지 슬라이싱|
- 인덱스 : 데이터의 순서 또는 위치를 나타냄
    - 0부터 시작
    - 형식 : 변수명[인덱스]
    
## 4.3 함수로 문자열 처리하기
- 파이썬에서는 문자열 다루기 쉽게 함수를 제공
    | 함수 | 의미 |
    |------|------|
    | lower() | 문자열을 소문자로 변환 |
    | upper() | 문자열을 대문자로 변환 |
    | islower() | 문자열이 소문자인지 확인 |
    | isupper() | 문자열이 대문자인지 확인 |
    | replace() | 문자열 바꾸기 |
    | index() | 찾는 문자열의 인덱스(없으면 오류 발생) |
    | find() | 찾는 문자열의 인덱스(없으면 -1 반환) |
    | count() | 문자열이 나온 횟수 | 

- 사용 방법 : 문자열(또는 변수).함수()

## 4.4 문자열 포매팅
- 문자열 포매팅 : 원하는 위치에 특정한 값(또는 변수)을 넣어서 하나의 문자열로 표현하는 방식
### 4.4.1 서식 시정자 사용하기
- 서식 지정자 : 자료형을 표현하는 방법, %뒤에 자료형을 나타내내는 문자가 붙음
    | 서식지정자 | 의미 |
    |----------|-------|
    | $d | 정수 |
    | $f | 실수 |
    | $c | 문자 |
    | $s | 문자열 |
- 형식 : print("문자열 서식지정자 문자열" %d)

### 4.4.2 format() 함수 사용하기
- 형식 : print("문자열{인덱스}문자열{인덱스}...".format(값1,값2,...)) or print("문자열{이름1}문자열2{이름}...".format(이름1=값1, 이름2=값2))

### 4.4.3 f-문자열 사용하기
- 문자열 앞에 f를 추가하면 문자열이 나오기 전에 정의한 변수의 값을 문자열에서 사용 가능(단,파이썬 3.6이상에서 사용 가능)
 - 형식 : print(f"문자열{변수명}문자열{변수명}...)

## 4.5 탈출 문자
- 탈출 문자 : "\"와 특정 문자를 함께 사용하여 문장 안에서 원하는 동작을 수행하는 방법
### 4.5.1 \n
- \n : 문자열 안에서 줄 바꿈할 때 사용
### 4.5.2 \"와 \'
- \" : 문자열 안에서 큰따옴표를 사용할 때 사용
- \' : 문자열 안에서 작은따옴표를 사용할 때 사용
### 4.5.3 \\
- \\  : 문자열 안에서 "\"를 사용할 때 사용
- r-문자열 : 문자열 앞에 r을 붙이면 문자열 내에 어떤 값이 있든지 무시하고 그대로 출력
### 4.5.4 \r
- \r : 커서를 맨 앞으로 이동, 이후 단어는 커서가 이동한 후 덮어쓰기 처리
### 4.5.5 \b
- \b : 키보드의 백스페이스 역할
    - 커서를 한 칸 앞으로 이동한 후 이후 오는 글자를 덮어 씀으로 덮어 쓰는 글자가 짧은 경우 원하는 단어가 완벽하게 삭제 되지 않을 수 있음
### 4.5.6 \t
- \t : 키보드의 'Tab'과 같이 여러 칸을 띄어 쓰는 역할
    - 터미널 기준으로 첫번째 글자로부터 8칸
