# 6. 제어문
## 6.1 조건에 따라 분기하기 : 조건문
### 6.1.1 조건이 하나일 때: if문
- 분기 : 조건에 따라 동작이 달라질 때
- 조건문 : 코드에서 분기할 때 사용
- 형식
    ```python
    if 조건 : 
        실행할 명령
    ```
- 조건을 만족할 때 실행 되는 코드는 if 위치 기준으로 4칸 들여쓰기를 해야하며 들여쓰기를 한 코드는 모두 실행. if 종료 후 실행할 코드의 경우 if와 같은 위치에 코드 작성 

### 6.1.2 조건이 여러 개일 때 : elif 문
- elif : 조건이 어려개인 경우 두번째 조건문부터 사용
- 형식
    ```python
    if 조건1 : 
        실행할 명령1
    elif 조건2 :
        실행할 명령2
    elif 조건3 :
        실행할 명령
    ...  
    ``` 

### 6.1.3 모든 조건이 맞지 않을때 : else 문
- else : 조건이 모두 해당하지 않을 때 사용
- 형식
    ```python
    if 조건1 : 
        실행할 명령1
    elif 조건2 :
        실행할 명령2
    elif 조건3 :
        실행할 명령
    ...  
    else :
        실행할 명령n # 앞의 조건에 모두 해당하지 않을 때 실행
    ``` 

### 6.1.4 input()으로 값 입력받아 비교하기
- input() : 프로그램을 실행했을 때 사용자로부터 어떤 값을 입력받는 용도로 사용
    - 값을 입력하고 enter를 누르면 **문자열 형태로 변수에 저장**
    - 정수형을 입력받는 경우 int()함수로 감싸 정수형으로 변환하여야 함

## 6.2 같은 일 반복하기 : 반복문
- 반복문 : 어떤 일을 반복해서 수행하고자 할 때 사용
### 6.2.1 범위 안에서 반복하기 : for 문
- 형식 : 
    ``` python
    for 변수 in 반복 대상:
        실행할 명령1
        실행할 명령2
        ...
    ```
- range()
    ``` python
    - range(숫자) # 0부터 지정한 숫자 직전까지 연속한 정수 반환
    range(시작 숫자, 끝 숫자) # 시작숫자부터 끝 숫자 직전까지 연속한 정수 반환
    range(시작 숫자, 끝 숫자, 간격) # 시작숫자부터 끝 숫자 직전까지 간격만큼 띄운 숫자 반환 
    ``` */## 6.2.2 조건을 만족할 동안 반복하기: while 문  
- 형식
    ```python
    whlie 조건:
        실행할 명령1
        실행할 명령2
    ```  
- 무한 루프 : while을 종료하는 조건이 없어 무한으로 실행되는 경우
    - 탈출 방법 : ctrl + c

### 6.2.3 흐름 제어하기: continue와 break
- continue : continue 이후 명령들을 실행하지 않고 다음 반복 대상으로 이동
- break : 반복문을 즉시 탈출

### 6.2.4 for 문 한 줄로 작성하기
- 형식
    ```python
    [동작 for 변수 in 반복 대상]
    ```
    - 동작에서 사용되는 변수명과 변수에서 사용되는 변수명이 같아야 원하는 결과가 나올 수 있음
