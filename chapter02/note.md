# 2장 자료형과 변수
- 자료형 : 데이터의 형태

## 2.1 숫자 자료형
- print() : () 안 내용(값)을 화면에 출력하라는 의미 
- 숫자 자료형 : 숫자로된 데이터, ex) 1,2,3, 1.2, 3.14

- 수식
    - "*" : 곱셈
    - "/" : 나눗셈

## 2.2 문자열 자료형
- 문자열 : 한글이나 알파벳 같은 문자
    - 작은 따옴표('')나 큰따옴표("")로 감싸야함
        - 작은 따옴표는 작은따옴표, 큰따옴표는 큰따옴표 끝내야함.
    - 문자열 \*(곱하기)를 하면 곱한 숫자만큼 문자열을 반복


## 2.3 불 자료형
- 불(boolean) : 참(True)과 거짓(False)만 가지는 값
    - 값을 넣으면 그대로 출력됨
- not : 값을 부정하는 연산자
    - True -> False, False -> True로 변경됨

## 2.4 변수
### 2.4.1 변수 정의하기
- 변수 : 어떤 값을 저장하는 공간
    - 형식 : 변수명 = 값
- "=" : 대입연산자
    - 값을 대입
- "변수를 정의한다" : 변수를 만들고 값을 대입해 저장하는 과정
- 변수명 규칙(naming convention)
    - 변수명에는 소문자(a-z), 대문자(A-Z), 숫자(0-9), 언더바(_)를 사용
    - 변수명은 **대소문자 구분**
    - 변수명은 숫자로 시작할 수 없고, 소문자, 대문자, 언더바로만 시작 가능
    - 키워드(파이썬에서 사용이 예약된 문자열)는 사용 불가

### 2.4.2 변수 사용하기
- 형식 : "문자열" + 변수 + "문자열"
- "+" 연산자 : 문자열과 변수를 연결해주어야함
    - "+" 연산자로 연결 시 자료형이 같아야 함.
- ","로 연결
    - "+" 연산자와 동일하게 문자열을 바꾸나 두가지가 다름
        - 형변환하지 않아도 됨
        - 값과 값 사이에 빈칸을 하나 포함한다.

### 2.4.3 형변형
- 형변환 자료형을 바꾸는 것
    - str() : 값의 자료형을 문자열로 변경
    - int() : 값의 자료형을 정수형으로 변경
    - float() : 값의 자료형을 실수형으로 변경
- type() : 자료형을 확인할 때 사용
    - 결과 : <class,'자료형'>

### 2.4.4 변수를 사용할 때 유의할 점
- 사용하기 전에 정의
- 마지막으로 저장된 값을 사용

### 2.5 주석
- 주석 : 코드에 설명을 표기할 때 사용
    - "#"을 붙이면 "#" 이후에 오는 내용은 주석처리(한줄만)
    - 주석으로 처리하고 싶은 문장의 시작과 끝부분에 "따옴표 3개(큰따옴표 또는 작은따옴표)를 사용하면 "따옴표 3개" 사이는 전체 주석처리
